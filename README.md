## 🌟 Overview
A premium, glassmorphic Progressive Web App (PWA) designed specifically for **Chindhamani Cycle Traders**. This system handles everything from daily billing and real-time inventory management to deep sales analytics and automated Excel reporting.

> [!TIP]
> This app is fully **PWA-enabled**. You can "Install" it on your Android or iOS device to use it just like a native mobile app!

---

## 🚀 Key Features

### 💎 Premium UI/UX
- **Glassmorphic Design**: Modern, sleek interface with vibrant gradients and smooth micro-animations.
- **Responsive Layout**: Works perfectly on Desktop, Tablets, and Mobile phones.

### 🧾 Smart Billing System
- **Real-time Stock Update**: Stocks are automatically deducted the moment a bill is generated.
- **Searchable Inventory**: "Flipkart-style" search bar to find products instantly.
- **Instant Invoices**: Generate professional invoices that can be downloaded as PDF or shared directly as an image.

### 📦 Inventory Monitoring
- **Automatic Low-Stock Alerts**: Visual warnings and notifications when items are running low.
- **Categorized Management**: 28x1.5, 26x1.5, MTB, Kids, and more.
- **Bulk Stock Updates**: Update wholesale and retail prices in seconds.

### 📊 Advanced Reporting
- **All Stocks Report**: Full tabular view of entire inventory with "Actual vs Retail" pricing.
- **Sales Dashboard**: Real-time charts (Line & Doughnut) for Daily, Weekly, Monthly, and Yearly revenue.
- **One-Click Export**: Download any report directly into **Excel-compatible CSV** files.

### 🔒 Secure Role-Based Access
- **Developer Mode**: Access to full development data.
- **Admin Mode**: Completely isolated, fresh production environment.
- **Daily Auto-Logout**: Sessions expire every midnight for maximum security.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+).
- **Charts**: [Chart.js](https://www.chartjs.org/)
- **Database**: [Firebase Realtime Database](https://firebase.google.com/products/realtime-database)
- **Deployment**: [Vercel](https://vercel.com/)
- **Libraries**: `html2canvas` for invoice sharing.

---

## 📦 Installation & Setup

1. **Clone the Repo**
   ```bash
   git clone https://github.com/your-username/cycle-shop-billing.git
   ```

2. **Configure Firebase**
   - Open `js/firebase-config.js`.
   - Replace the placeholders with your actual Firebase API keys.

3. **Deploy**
   - Push to GitHub and connect your repository to **Vercel**.
   - Vercel will automatically handle the deployment using the included `vercel.json`.
